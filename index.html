About Me

Saya adalah seorang Web Developer dengan latar belakang kuat di pengembangan aplikasi berbasis Java dan pengalaman dalam membangun website perusahaan menggunakan berbagai teknologi seperti Spring Boot, Thymeleaf, ZKoss, dan Oracle.

Selama beberapa tahun terakhir, saya telah menangani berbagai proyek web, baik untuk kebutuhan internal perusahaan maupun untuk klien eksternal, mulai dari pengembangan front-end yang responsif hingga integrasi sistem back-end yang kompleks.

Saya terbiasa dengan praktik DevOps, termasuk implementasi CI/CD menggunakan GitLab dan Docker, serta mampu bekerja secara mandiri maupun dalam tim. Saya selalu berusaha menciptakan aplikasi yang efisien, aman, dan mudah digunakan.

Di luar pekerjaan utama, saya juga aktif mengeksplorasi teknologi baru dan terbuka untuk kolaborasi dalam proyek freelance maupun open-source.
